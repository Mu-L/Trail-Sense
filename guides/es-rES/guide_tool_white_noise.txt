La herramienta Ruido Blanco se puede usar para ayudarte a dormir mientras acampas o haces senderismo con mochila.

## Uso
Pulsa el botón de ruido blanco para activar o desactivar el ruido blanco. Se mostrará una notificación mientras el ruido blanco se esté reproduciendo y puedes pulsarla en cualquier momento para detener el ruido blanco.

## Selección de sonido
Puedes elegir el sonido preferido para dormir usando el menú desplegable debajo del botón. Esta selección se guardará para usos futuros de la herramienta de ruido blanco.

## Temporizador de sueño
Puedes configurar un temporizador de sueño para detener automáticamente el ruido blanco después de un cierto tiempo. Para configurar el temporizador de sueño, activa el interruptor del temporizador y escribe la cantidad de tiempo que quieres que el ruido blanco suene. El temporizador empezará a contar regresivamente tan pronto como inicies el ruido blanco.

## Acción rápida
Puedes habilitar la acción rápida Ruido blanco en los ajustes para la pestaña en la que la quieras.

Para usar la acción rápida, pulsa el botón de acción rápida Ruido Blanco para activarlo o desactivarlo.
