A ferramenta Anotações pode ser usada para escrever anotações com acesso offline.

## Criar ou editar uma nota
Para criar, visualizar ou editar uma nota:

1. Pressione o botão '+' no canto inferior direito para criar uma nota, ou toque em uma existente na lista.
2. Você pode editar o título e/ou o conteúdo da nota.
3. Pressione o botão de marca de verificação no canto inferior direito para salvar.

## Excluir uma nota
Para excluir uma nota:

1. Toque no ícone de menu ao lado direito da nota na lista.
2. Pressione 'Excluir'.

## Compartilhar uma nota (código QR)
Para compartilhar uma nota como código QR:

1. Toque no ícone de menu ao lado direito da nota na lista.
2. Pressione 'Código QR'.
3. Escaneie o código QR com outro dispositivo para importar a nota, ou faça uma captura de tela para uso posterior.

Nota: O código QR só pode transferir os primeiros 1000 caracteres de uma nota.

## Ação rápida
Você pode ativar a ação rápida de Anotações nas configurações da aba desejada.

Para usar a ação rápida, toque no botão de ação rápida de Anotações para criar uma nova nota.
